# MacroPlate Meal Prep Applications

Clean workspace scaffold for MacroPlate Meal Prep application workflows.

## Structure

- backend: Node.js + TypeScript Express API
- frontend: static starter UI
- shared: shared docs/types placeholders
- docs: project documentation

## Quick Start

1. Install dependencies:

```bash
npm run install:all
```

2. Start backend in dev mode:

```bash
npm run dev
```

3. Open frontend starter page:

Open frontend/index.html in your browser.

## Backend Endpoints

- GET /health
- GET /api/applications
